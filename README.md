### This mod contains two separate mods:
- 🐾 Pawmarks: Replaces all marks of distinction with paw prints.
- 🎨 Boykisser Mod: Redesigns tank camouflages.

### Installation:
- Download archive
- Unzip it to to `WoT/mods/{verison}/`
